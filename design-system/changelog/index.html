#Change log of Eufemia

You may also be interested in details about releases and have a look at the GitHub Releases for versioning of the dnb-ui-lib.

#August, 24. 2019

  • Read more about the dnb-ui-lib v4.10 release
  • New component in the dnb-ui-lib: GlobalStatus
  • Updated FormStatus with new text color and new icon
  • Added link to example usage of a Table styles using react-table
  • Added info about Custom project Icons

#July, 21. 2019

  • Major Release of dnb-ui-lib: Read more about the v4 release
  • New component in the dnb-ui-lib: Space for easy spacing capabilities
  • New component in the dnb-ui-lib: Slider with add/subtract buttons

#June, 26. 2019

  • The dnb-ui-lib got new helpers for Spacing and Layout
  • Added first Demo App

#June, 19. 2019

  • New components in the dnb-ui-lib: ToggleButton as well as two layout components FormSet and FormRow

#June, 5. 2019

  • New corner radius (border-radius) on Dropdown and DatePicker: 4px (0.25rem)

#June, 3. 2019

  • New component in the dnb-ui-lib: Textarea. Got also a wider corner radius by 16px (1rem)

#May, 27. 2019

  • New components in the dnb-ui-lib: Checkbox and Radio button

#May, 19. 2019

  • New component in the dnb-ui-lib: Circular ProgressIndicator bar (spinner with animation)

#May, 8. 2019

  • New component in the dnb-ui-lib: DatePicker with range/period support

#April, 26. 2019

Changes to the default styles

  • DNB section helper class .dnb-section got two new default colors .dnb-section--emerald-green and .dnb-section--signal-orange.

#March, 22. 2019

Changes to the default styles

  • New Primary and Secondary Icons where added (link_out, pencil, bubble, save)
  • Enhanced the behavior of the Input Component with a entered text (dirty) state, changing the typography color to Emerald Green

#March, 12. 2019

Changes to the default styles

  • Toggle Switch got new states and look & feel
  • New Primary Icons where added (information)
  • Overall Component enhancements

#March, 6. 2019

Changes to the default styles

  • ErrorMessage (FormStatus) got a smaller text size (14px)
  • Several "v1" walk though, into detail changes

#February, 21. 2019

Changes to the default styles

  • Anchor (Text Link): Has now white background color on focus state
  • Tables how now always a bottom border
  • Button got a new look for all states
  • Input got a new look for all states
  • Dropdown got a new look for all states

UI library

  • Breaking change: Styles got new structure. You have now to declare the HTML Elements with a class

#February, 14. 2019

  • Anchor (Text Link): Removed underline on hover and active states
  • Tables (the default once) has now always a border line on the bottom

#January 2019

Changes to the default styles

  • Change from 1px to 2px outline on input for error state
  • Signal Orange is replaced by Emerald Green for focus state
  • Tab and StepIndicator (Progress bar) have now Sea Green Alt underline
  • The Switch component has now an outline of 2px
  • Hover on Tabs looks more like an anchor
  • Color change: Sea Green 4% -> Mint Green 12%
