## Typographic rules

The following are the default typographic rules for basic typographic elements such as headings and body text.

### Margins

#### Margins in general

Blocks of text (headings and body etc.) do not have TOP, LEFT or RIGHT margins by default. This is usually set by the containing element and based on context.

#### Bottom Margins

Blocks of text (headings and body etc.) have a default BOTTOM margin of 1rem ( 16px ). This can be adjusted depending on context. Make sure to stick to the Eufemia spacing system when adding or removing bottom margins.
